# cinema

Цель разработки: Практика React.

Были реализованы: 
  1. Роутинг
  2. Поиск фильмов с дропменю.
  3. Бесконечная прогрузка фильмов
  4. Регистрация, авторизация
  5. Кастомные хуки

## Запуск

1. Установить зависимости:

    ```bash
    npm install
    ```

2. Запустить приложение:

    ```bash
    npm run dev
    ```
